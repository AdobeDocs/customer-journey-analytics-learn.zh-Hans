# Customer Journey Analytics 是什么？

通过 Customer Journey Analytics (CJA)，可使用任何在 Adobe Experience Platform 中对您可用的数据分析客户在各个渠道中的历程，并利用 Analysis Workspace 的强大功能迅速获得见解。
