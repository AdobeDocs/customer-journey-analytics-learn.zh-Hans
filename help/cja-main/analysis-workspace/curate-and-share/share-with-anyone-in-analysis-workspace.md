# Analysis Workspace 中的“与任何人共享”

了解Workspace中的“与任何人共享”功能。 您可以使用此功能向无权访问Workspace或Customer Journey Analytics的用户（包括组织外部的收件人）授予对Adobe Analytics项目的只读访问权限。
