# 在数据视图中绑定Dimension

绑定维度是Customer Journey Analytics中一项令人兴奋的功能，它使您能够采用一个维度并将其连接到另一个维度，以便更精细地分配持久性。 我们经常会遇到以下情况：在用户的整个历程中，我们会收到不同的值，因此需要决定如何分配这些成功量度。 “绑定”维度是帮助解决某些关键情景的方法。
