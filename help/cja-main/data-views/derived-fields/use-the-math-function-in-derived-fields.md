# 在Customer Journey Analytics的派生字段中通过使用math函数来调整指标

在本视频教程中探索使用数学函数在 Customer Journey Analytics 中创建派生字段的潜力。 学习通过从总计中删除不需要的组件（如税）来调整指标，并了解使用派生字段相对于计算量度的优势，包括独占的数据视图功能，如重复数据删除、分段和归因。 获得创建符合您分析需求的量度定制指标的灵活性，增强您数据驱动的决策能力。
