# [!UICONTROL 数据视图]中的[!UICONTROL 归因]设置

每个维度都应该具有分配设置（以确定哪个值获得转化点数）和有效期限设置（该值持续多久），以便系统知道如何为维度值分配值。 同样，结合使用的量度也应该具有归因模型和回溯时段设置。 本视频指导您完成这些非常重要的设置。
