# 为 Customer Journey Analytics 配置数据视图的概述

在 Customer Journey Analytics 中，数据视图类似于 Adobe Analytics 中的虚拟报告包；它们允许您配置进入 CJA 的数据，以便对您的报告和分析发挥最大效用。本视频概述了如何为 Customer Journey Analytics 配置数据视图。
