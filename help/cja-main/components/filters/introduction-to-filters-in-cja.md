# 区段概述

区段根据属性和事件（如网站访客或呼叫中心参与）识别访客或客户的子集。 了解Customer Journey Analytics中区段生成器的关键组件。 有关 Customer Journey Analytics 的更多信息，请访问此[文档](https://experienceleague.adobe.com/en/docs/analytics-platform/using/cja-components/cja-segments/filters-overview){target="_blank"}。

>[!NOTE]
>
> 在Customer Journey Analytics界面中，[!UICONTROL 筛选器]已重命名为[!UICONTROL 区段]。
