# 用于Customer Journey Analytics的受众发布

Customer Journey Analytics允许您将分析中发现的受众发布到Adobe Experience Platform实时客户资料，以便使用Adobe Real-time Customer Data Platform激活区段，或使用Adobe Journey Optimizer实现更个性化的旅程编排。
