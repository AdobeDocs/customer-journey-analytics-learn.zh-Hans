# 实时报告

了解Adobe Customer Journey Analytics中的实时报表如何直接在Analysis Workspace中提供收入、转化和访问等关键量度。 此功能非常适合于重大促销、产品发布和历程验证，它使营销人员能够在每一分钟都非常重要时做出更快、以数据为导向的决策。
