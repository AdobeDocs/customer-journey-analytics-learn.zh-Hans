# 实时报告

了解Adobe Customer Journey Analytics中的实时报表如何直接在Analysis Workspace中提供收入、转化和访问等关键量度。 此功能非常适合于主要促销活动、产品发布和历程验证，它使营销人员能够在每一分钟都非常重要时做出更快、数据驱动的决策。

有关更多信息，请阅读该[文档](https://experienceleague.adobe.com/en/docs/analytics-platform/using/cja-components/real-time-reporting/real-time){target="_blank"}。
