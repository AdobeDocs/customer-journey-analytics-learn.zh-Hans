# 协助管理人员访问移动记分卡

协助管理人员使用 [!DNL Analytics] 仪表板移动应用程序访问在 [!DNL Customer Journey Analytics] 中创建的移动记分卡。  移动记分卡可提供随时访问关键绩效指标和详细视图或细分内容的权限。
