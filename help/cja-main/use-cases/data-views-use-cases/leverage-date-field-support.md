# 利用日期字段支持

日期字段支持增加了在 Customer Journey Analytics 中处理日期和日期时间字段的能力。您可以选择日期或日期时间字段的显示方式，以及在包含/排除功能中使用特定于日期的逻辑运算符来过滤显示的内容。
