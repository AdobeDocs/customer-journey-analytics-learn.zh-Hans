# [!DNL Customer Journey Analytics]的远程MCP服务器概述

了解如何将[!DNL Claude]、[!DNL ChatGPT]或[!DNL Cursor]等AI工具连接到Adobe Customer Journey Analytics (CJA) MCP服务器，以使用自然语言浏览数据视图、执行报表并生成可用于执行处理的KPI见解。 此演示展示了远程MCP服务器如何在Adobe Experience Platform中启用可扩展、受管理的分析工作流和自动报告。

有关更多信息，请参阅 Content Analytics [文档](https://developer.adobe.com/analytics-mcp/docs/cja/){target="_blank"}。
