# 查看、疑难解答和修改连接设置

连接管理器允许您检查连接数据集摄取的状态。 此界面还会通知您何时有数据可在工作区中开始分析。 了解如何识别由于配置错误导致的数据差异、管理数据导入日期以及从连接中删除数据集。
