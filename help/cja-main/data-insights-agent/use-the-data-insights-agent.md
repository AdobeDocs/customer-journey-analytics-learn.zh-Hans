# 使用 [!DNL Data Insights Agent]

借助自然语言提示分析数据，自动生成自由格式表、可视化图表和数据对比，从而减少对数据分析师的依赖。 这款 AI 驱动代理可快速洞察趋势与绩效，助您即时向利益相关者分享可执行的洞察结果。
