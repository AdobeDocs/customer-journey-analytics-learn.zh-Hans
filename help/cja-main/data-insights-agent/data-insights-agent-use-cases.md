# [!DNL Data Insights Agent]用例

使用自然语言查询、实时可视化和自由格式表简化分析。 提升营销策略制定、营销活动管理和业务绩效报表的效率。 这一由AI提供支持的解决方案使营销人员能够即时获得洞察，并释放分析人员以执行战略任务，增强客户体验和主动决策。
