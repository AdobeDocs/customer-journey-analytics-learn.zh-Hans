# [!DNL Data Insights Agent] 简介

由 Adobe 的 Experience Platform Agent Orchestrator 提供支持的 [!DNL Data Insights Agent]，为营销人员和业务用户带来了客户体验编排方式的革新。 它利用生成式 AI，通过自然语言查询提供实时、个性化的洞察，打破传统分析师流程的瓶颈。

有关更多信息，请参阅 Data Insights 代理的[文档](https://experienceleague.adobe.com/zh-hans/docs/analytics-platform/using/cja-overview/cja-b2c-overview/data-analysis-ai){target="_blank"}。

## 简介

了解 Customer Journey Analytics 中的 [!DNL Data Insights Agent]。 这款 AI 驱动的解决方案可为营销人员和业务相关方提供即时、透明的洞察，消除延迟和大量培训成本，同时释放分析师的精力以专注于战略性任务。
