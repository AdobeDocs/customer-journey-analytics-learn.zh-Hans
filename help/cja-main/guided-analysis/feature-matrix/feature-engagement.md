# 功能参与度分析

了解如何在Customer Journey Analytics中使用&#x200B;**功能参与视图**。 此分析提供有关使用您的产品功能的频率以及使用这些功能的人员数量的见解。 同时比较多个功能以获得最佳优势。 这有助于您通过了解您的核心、能力、一次性和可疑功能做出投资决策。
