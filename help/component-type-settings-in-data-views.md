# 数据视图中的组件类型设置

在Customer Journey Analytics的“数据视图”中配置组件时，您可以选择字段是量度还是维度。 在许多情况下，字符串应为维度，数字应为量度。 但是，有一些很好的用例可将其切换。 看看！
